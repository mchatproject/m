M: Api reference.

/api/users/page/{num}

Serves user data for page {num}, you can use GET on this.

/api/users/

Serves ALL user data, insanely laggy. [You can also use get on this, You shouldnt though. You really shouldnt.]

/api/users/{user}

Serves {user}'s user data, you can use GET on this.

/api/

Useless.
